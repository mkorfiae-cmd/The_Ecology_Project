# **Интеллектуальная система визуальной детекции лесных пожаров «FireDetection»**
### 1. Проблема (контекст и актуальность)

Лесные пожары являются одной из ключевых угроз природным экосистемам, приводя к уничтожению лесных массивов, гибели животных, ухудшению качества воздуха.

**Основные проблемы существующего мониторинга:**

* Позднее обнаружение пожаров: на ранней стадии пожар часто проявляется только дымом.

* Ограниченность традиционных методов: спутниковые тепловые аномалии фиксируются с задержкой, а визуальный контроль ведётся нерегулярно.

* Человеческий фактор: ручной просмотр изображений и видео требует времени и ресурсов.

---

### 2. Решение (учебная IT-система «FireDetection»)

**FireVision** - учебный прототип интеллектуальной системы, использующей компьютерное зрение для автоматической детекции огня на изображениях с помощью предобученной модели машинного обучения.

Проект демонстрирует, как готовые ML-инструменты могут применяться для задач экологического мониторинга без сложной инфраструктуры.

---

### 3. Техническая концепция проекта
**Назначение системы**
* Анализ одного изображения (или кадра)
* Автоматическое обнаружение признаков пожара
* Визуализация результатов и текстовый вывод

**Основные компоненты**
| Компонент	 | Входные данные	| Функция |
| :--- | :--- | :--- |
| Детекция дыма и огня	| Изображение (URL или локальный файл)	| Предобученная модель компьютерного зрения выполняет object detection |
| Обработка результатов	| Выход модели	| Извлечение классов, координат и confidence |
| Визуализация	| Результаты детекции	| Отрисовка bounding box и подписей |
| Аналитический вывод	| Детекции	| Подсчёт объектов и формирование текстового сообщения |
